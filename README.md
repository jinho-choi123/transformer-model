# Transformer Model

## Getting Started
**setup venv and install python packages using [uv-python](https://astral.sh/blog/uv)**
```shell
# setup venv for python
$ uv venv
$ source .venv/bin/activate
$ uv pip install -r requirements.txt

# install script for pre-commit
$ pre-commit install

# run jupyter lab
$ make run
```
